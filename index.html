Hi, my dear Dandelion! I'm really happy to see you here. Let's get started.

Since you already have administrator privileges, you can install and delete applications and do many other things. By the way, we will also learn common commands for Linux.

Ok, open the terminal and find out what the python version is installed.

$ python3 --version
Python 3.8.10
If you see something like that, congrats! You already have installed Python.

$ python3 --version
-bash: python3: command not found

But if you see the phrase "python: command not found", it stands you don't have any python versions on your laptop, and you have to install it.

Follow this link "How To Install Python 3 and Set Up a Programming Environment on Ubuntu 22.04" if you need to install Python.

Move on, create a new folder in your $HOME (this is your private directory in the Linux system. "Folder" and "Directory" are the same. Notice that $HOME is a variable. Type "echo $HOME" in command line.)

lyaki4@ubuntu:~$ echo $HOME # This is my home folder
/home/lyaki4

lyaki4@ubuntu:~$ pwd # "Print Working Directory" - where I am
/home/lyaki4

lyaki4@ubuntu:~$ mkdir work # Create folder "work" in the current directory

lyaki4@ubuntu:~$ cd work # Change current directory to "$HOME/work"

lyaki4@ubuntu:~/work$ pwd # "Print Workin Directory" - where I am
/home/lyaki4/work

lyaki4@ubuntu:~/work$ mkdir project_01 && cd $_ # Create folder "project_01" in "work" and change directory to it
lyaki4@ubuntu:~/work/project_01$

symbol # is a beginning of comments.

$ sudo apt update # Do that always before installation of a new software
$ sudo apt install -y python3-pip # You are going to install package pip. PIP is a package manager for Python packages.
$ sudo apt install -y build-essential libssl-dev libffi-dev python3-dev # These are few more packages for work with Python
Since today you know following:
echo - display a line of text
pwd - print name of current/working directory
mkdir - make directories
cd - Change the shell working directory
sudo - execute a command as another user
apt - commandline interface for the package management system
